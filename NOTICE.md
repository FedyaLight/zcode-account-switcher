# Notices

This repository contains a native macOS implementation of a ZCode account
switcher.

The account snapshot model, credential format knowledge, provider routing
behavior, quota endpoint behavior, and some import/export compatibility choices
were informed by the MIT-licensed
[smartlizi/zcode-account-switcher](https://github.com/smartlizi/zcode-account-switcher)
project.

The implementation in this repository is written in Swift and uses a different
runtime, application architecture, UI, process management layer, build system,
and test suite.

ZCode and related product names, icons, and marks belong to their respective
owners. This project is not affiliated with or endorsed by ZCode.
